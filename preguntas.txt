variable: es un tipo de dato que puede tomar cualquier valor, puede ser definido por uno o ser el resultado de una operacion. Podemos cambiar el valor de la variable las veces que necesitemos inclusive si necesitamos cambiar el tipo de variable.
strings: se refiere a los datos de tipo texto, deben escribirse entre comillas para que funcionen.
funciones: son rutinas que permiten hacer bloques de tareas que necesitamos hacer varias veces y asi reducir la cantidad de codigo que escribimos. Las definimos usando la palabra function luego un espacio seguido del nombre que le vamos a dar a la funcion, luego entre parentesis podemos colocar las variables que se usaran en la funcion, despues entre corchetes se colocara el argumento que son las diferentes actividades que se ejecutaran en la funcion. Si utilizamos la palabra return le indicamos que queremos que devuelva como resultado la funcion.
declaraciones: son las instrucciones que le damos al interprete para que haga las tareas que necesitamos. Una de las declaraciones que podemos utilizar es if que se refiere a ejecutar una tarea si una condicion que propongamos es verdadera o ejecutar otra si es falsa. Para utilizarla la llamamos escribiendo if seguido de la condicion que se va a evaluar entre parentesis y luego entre corchetes escribimos las tareas que ejecutara si la condicion propuesta es verdadera. Luego escribimos else seguido de corchetes para escribir las actividades que se ejecutaran si la condicion no es verdadera. 
Valores Booleanos: son unos tipos de datos que solo pueden tener dos valores posibles true(verdadero) o false (falso). Podemos definir variables del tipo booleano dandole un nombre como cualquier otro tipo de variable.
